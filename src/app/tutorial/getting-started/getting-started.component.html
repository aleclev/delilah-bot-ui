Getting started with Delilah

Learn about Delilah and how to use various features.

Features of DelilahBot:

Group Events (LFG)

Allows users to create interactive group event posts:




All Sherpa Run users can use the Join and Leave buttons.

Group owners have access to a few more options:

Summon Group: Sends a private message to all members of your group.

Alert: Sends a private message to all users who are subscribed to your activity (see notification module).

Delete: Allows the group owner to delete the group.

You can create Event Groups here, or by using the '/lfg create' command.

 Notification

Allows users to manage which Group Event posts they want to be notified for. You can also choose to block notifications from certain users.

Note: You will not receive notifications if you are offline or have do not disturb selected in Discord.

 Dictionary (Word)

Allows users to create Word/Definition pairs that can be retrieved later. This is useful if you need to access information frequently. For instance, sherpas use this to display learning aid for raids.

You may have heard of Discord bots. These are programs that users can interact with through the Discord app. Typically, bots respond to slash commands to do various things. Some bots, like Delilah, include a web interface for better accessibility.

1st step - Registering

For many Delilah features, a user profile must be registered.

This can be done by logging into this website, or by executing the '/register discord' slash command.

To use slash commands, please go to Sherpa Run and visit the #commands channel. From there, type '/' to see all commands. To execute the command, click on it and press enter.




2nd step - Understand slash commands

Slash commands have multiple parts:

1: Name. Commands can be filtered by their name.

2: Description. A brief description of what the command does. (When focusing an argument, this will be the argument's description instead)

3: Arguments. These are values used to execute some commands. Arguments can be required or optional. All required arguments must be provided for the command to be executed.

Once you've selected a command and entered its arguments, press enter to execute it.

3rd - Explore

This should give you the basic knowledge you need to interact effectively with Delilah's functionalities. Now you can log in and explore this website or consult the full list of slash commands on Discord.
